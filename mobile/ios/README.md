# VaultZero iOS

Placeholder for the iOS native app using Swift + SwiftUI.

## Implementation: Day 12-13

This will include:
- `CredentialProviderExtension` for system-level AutoFill
- Face ID / Touch ID via `LocalAuthentication`
- Keychain storage with `kSecAttrAccessibleWhenUnlockedThisDeviceOnly`
- CryptoKit for AES-256-GCM

The Xcode project will be created when we reach Day 12.
